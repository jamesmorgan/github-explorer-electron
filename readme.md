# Github Explorer Electron

## Dev

```bash
npm install
```

### Run

```bash
npm start
```

### Build a distributable

```bash
npm install electron-packager -g
```
& 
```bash
npm run build
```

* Currently only builds for `OSX` & `Linux` using [electron-packager](https://github.com/electron-userland/electron-packager).
